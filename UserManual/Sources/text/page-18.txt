4.24 PROGRAMMING Maxx can be programmed to move forward or in reverse.
BODY MOVEMENTS

*To Program Forward Or Reverse Movement:

1, Press the <PROGRAM> key to enter the Program Mode.

2. Press the <DRIVE FORWARD=> or <DRIVE REVERSE> key.

3. The display will read ‘'F__"' [forward] or ““b__” [reverse]. This is a
PROMPT indicating that Maxx expects a number, from 0 to 255,
specifying how many inches forward or backward he should move. The
number or numbers you enter, multiplied by approximately one inch,
will specify how far Maxx will move.

4, After the number keys have been pressed, press the <ENTER> key to
complete and store the program step.

5. The display will now show the number of the next program step,
indicating that Maxx is ready for further programming.

NOTE: When the program is executed, Maxx will move ot his high
speed, There is no shifting from the low to the high speed in the
Program Mode.
