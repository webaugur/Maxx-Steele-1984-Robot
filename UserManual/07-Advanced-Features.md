Program Execution Mode, Maxx goes to the next instruction before

functions at the same time. For example, Maxx can speak, play music,
walk and move his arms and claw all ot the same time. (Obviously,

#### 5.42 Pausing

In the Execute Mode, the `<NOTE REST/WAIT>` key can be used to make Maxx’s
program pause. This is useful for program refinement. For example,
suppose you programmed Maxx and, after running the program,
found that the sequence of actions or responses isn't quite correct after
step 10. Run the program again; but use the `<NOTE REST/WAIT>` and
`<ENTER>`keys to make Maxx pause at step 10. The corrections to the
program can now be put in by using the Edit Mode (see 5.41 above).
